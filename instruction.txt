1. Перейти в директорию с проектом
2. Установить виртуальное окружение и активировать его

Установка виртуального окружения
====================================
# Windows
python -m venv venv

# Linux
python3 -m venv venv

Активация виртуального окружения
====================================
# Windows
.\venv\Scripts\activate

# Linux
source venv/bin/activate


3. Установить список зависимостей с помощью файла requitements.txt c помощью команды ниже
(При запущенном виртуальном окружении)

pip install -r requirements.txt.


4. Перейти в папку /mailmanage и запустить приложение на локальном сервере через команду 

 python manage.py runserver

http://127.0.0.1:8000/


(Приложение реализовано на python 3.7.4. В случае отсутствия его на пк, нужно также установить https://www.python.org/downloads/)
